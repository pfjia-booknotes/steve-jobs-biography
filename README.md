# 乔布斯传记读书笔记

这是一个关于沃尔特·艾萨克森所著《乔布斯传》的读书笔记项目，按照书的章节结构系统整理了乔布斯一生的重要事件、性格特点和个人思考。

## 项目结构

### 第一部分：起源（第1-9章）

**一、成长与觉醒（第1-4章）**
- 第一章 童年：被领养的经历、硅谷成长环境、早期性格形成
- 第二章 奇特的一对：两个史蒂夫：沃兹尼亚克与乔布斯的友谊、蓝盒子的故事
- 第三章 出离：克里斯安·布伦南、里德学院、罗伯特·弗里德兰、退学
- 第四章 雅达利与印度：雅达利工作经历、印度禅修之旅、打砖块游戏开发

**二、苹果公司诞生（第5-9章）**
- 第五章 AppleI：慈爱的机器、家酿计算机俱乐部、苹果诞生、车库工厂
- 第六章 AppleII：集成产品、迈克·马库拉、里吉斯·麦肯纳、一鸣惊人、迈克·斯科特
- 第七章 克里斯安和丽萨：被遗弃者的故事
- 第八章 施乐和丽萨：新宝贝、施乐PARC、"伟大的艺术家窃取灵感"

## 文件命名规范

### 数字前缀规则
- 一级文件夹：01_第一部分、02_第二部分...
- 二级文件夹：01_一、成长与觉醒、02_二、苹果公司诞生...
- 三级文件夹：01_第一章、02_第二章...
- 四级文件：01_文件名、02_文件名...

### 写作格式要求
- 每个笔记文件包含两部分：【故事梗概】和【对乔布斯的影响/性格体现】
- 文件开头直接从"### 故事梗概"开始，不包含章节标题
- 每小结总结后的汉字个数小于200字

## 写作格式示例

```markdown
### 故事梗概

1. 事件描述
2. 关键细节

### 对乔布斯的影响/性格体现

1. 影响分析
2. 性格体现
```

## 关于本书

《乔布斯传》是沃尔特·艾萨克森所著的史蒂夫·乔布斯唯一授权传记。艾萨克森历时两年，与乔布斯进行了40多次深入交谈，走访了乔布斯身边的100多位朋友、同事和竞争对手，呈现了一个立体而真实的乔布斯形象。

这本读书笔记项目旨在系统梳理乔布斯一生的重要事件，深入分析其性格特点，探讨其对科技行业和社会的深远影响。
