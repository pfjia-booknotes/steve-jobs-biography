# 乔布斯与阿甘

## 引言

乔布斯和阿甘，两个看似毫无关联的名字，却有着惊人的相似之处。一个是改变世界的科技巨人，一个是电影虚构的传奇人物。他们的故事都告诉我们：起点不能决定终点，被遗弃的孩子同样可以创造奇迹。

## 第一部分：人生轨迹

### 一、阿甘的人生轨迹

1. **1940年代-1950年代：阿拉巴马州的童年**
   - 1945年：出生于阿拉巴马州绿bow镇，智商只有75，腿部需要矫正器
   - 1945年：被父亲遗弃，与母亲相依为命
   - 1955年：在校车上遇见珍妮，开始了一生的友谊

2. **1960年代：上大学与越战**
   - 1964年：进入阿拉巴马州大学
   - 1966年：成为橄榄球明星
   - 1967年：被应召入伍，参加越南战争
   - 1969年：冒着枪林弹雨救回多名战友，获得荣誉勋章

3. **1970年代：乒乓球冠军与捕虾船长**
   - 1970年：退役后开始练习乒乓球
   - 1971年：成为全国冠军，作为乒乓外交使者访问中国
   - 1972年：见到总统尼克松
   - 1975年：买捕虾船，成为捕虾船船长

4. **1980年代：跑步穿越美国与平静生活**
   - 1978年：在华盛顿纪念碑前意外重逢珍妮
   - 1979年：开始跑步
   - 1981年：三年多横跨美国多次，成为全国知名人物
   - 1982年：珍妮因病去世，独自抚养女儿小珍妮

### 二、乔布斯的人生轨迹

1. **1955年：被领养的孩子**
   - 出生于旧金山，一周后被保罗和克拉拉·乔布斯领养
   - 在硅谷长大，对电子学产生浓厚兴趣

2. **1970年代：创立苹果**
   - 1974年：在雅达利工作，随后去印度灵修
   - 1976年：与沃兹尼亚克创立苹果公司，推出Apple I
   - 1977年：推出Apple II，改变个人电脑行业

3. **1980年代：辉煌与陨落**
   - 1984年：推出麦金塔电脑
   - 1985年：被逐出苹果，创立NeXT和皮克斯

4. **1990年代：回归与重建**
   - 1996年：苹果收购NeXT，乔布斯回归
   - 1998年：推出iMac
   - 2001年：推出iPod，改变音乐产业

5. **2000年代：重新定义手机**
   - 2007年：推出iPhone，颠覆手机行业
   - 2010年：推出iPad，改变平板市场
   - 2003年：被诊断出胰腺神经内分泌肿瘤

6. **2011年：传奇落幕**
   - 8月辞去苹果CEO职位
   - 10月5日去世，享年56岁

## 第二部分：对比分析

### 一、相似之处

1. **被遗弃的童年**
   - 乔布斯出生一周后被领养，阿甘因智商问题被父亲遗弃
   - 两人都有着强烈的证明自己的欲望
   - 起点不能决定终点

2. **时代的"宠儿"**
   - 阿甘"恰好"参加了越战、乒乓外交，见证猫王和约翰·列侬的相遇
   - 乔布斯"恰好"赶上个人电脑革命、智能手机革命和数字娱乐革命
   - 机会总是留给有准备的人

3. **对爱情的执着**
   - 阿甘对珍妮一生守候，不离不弃
   - 乔布斯与劳伦娜几十年婚姻，最终在生命尽头眷恋家庭
   - 两人都最终找到了属于自己的"那个人"

4. **嬉皮士文化与反主流**
   - 珍妮是反主流文化的代表，留长发、抽大麻、反对越战
   - 乔布斯深受嬉皮士文化影响，留长发、穿牛仔裤、追求精神自由
   - 1960-70年代的反主流思潮塑造了两代人的价值观
   - 阿甘的"单纯"与珍妮的"叛逆"形成对比
   - 乔布斯的"创新"与嬉皮士的"反叛"一脉相承
   - 两人都选择了不随波逐流，走自己的路

### 二、不同之处

1. **性格特质**
   - 阿甘：单纯，不理解复杂人际关系，凭本能和善良行事
   - 乔布斯：复杂，偏执、控制欲强，对完美有极致追求
   - 阿甘是"无心插柳柳成荫"，乔布斯是"刻意为之终成器"

2. **对待成功的态度**
   - 阿甘对成功看得很淡，跑步、捕虾都不是为了名利
   - 乔布斯对成功有强烈执念，享受改变世界的感觉
   - "无心之举"与"刻意为之"是最根本的差异

3. **面对困境的方式**
   - 阿甘面对困难是"接受"，不抱怨，默默继续前进
   - 乔布斯面对困难是"改变"，坚信可以改变现实
   - "阿Q精神"与"现实扭曲力场"形成鲜明对比

## 第三部分：启示

1. **关于起点**
   - 被遗弃的童年不是终点，而是新起点
   - 乔布斯和阿甘都用自己的方式证明了这一点
   - 重要的是如何面对，而非起点在哪里

2. **关于机遇**
   - 两人都不是刻意追求机遇，而是命运将他们推上舞台
   - 但"巧合"背后是他们已经准备好了
   - 机会总是留给有准备的人

3. **关于坚持**
   - 阿甘用善良和坚持赢得了人生
   - 乔布斯用激情和执着改变了世界
   - 无论性格如何，坚持都是成功的关键

4. **关于家庭**
   - 两人都最终成为了父亲
   - 阿甘选择独自抚养女儿，乔布斯经历了复杂的父女关系修复
   - 家庭是人生的归宿，无论过程如何

## 结语

乔布斯和阿甘的故事告诉我们：人生没有固定的轨迹。阿甘用善良和坚持赢得了人生，乔布斯用激情和执着改变了世界。两人都不是完人，但他们的故事激励着无数人去追寻自己的"人生巧克力"。正如阿甘所说："生活就像一盒巧克力，你永远不知道下一块是什么味道。"无论是被遗弃的孩子，还是迷茫的青年，都有可能创造出属于自己的奇迹。
