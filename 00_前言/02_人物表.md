# 人物表

## 一、家庭成员

### 养父母及养妹

1. **保罗·莱因霍尔德·乔布斯**(Paul Reinhold Jobs)：出生于威斯康星州，曾为海岸警卫队队员，跟妻子克拉拉于1955年领养了史蒂夫·乔布斯。

2. **克拉拉·哈戈皮安·乔布斯**(Clara Hagopian Jobs)：父母为亚美尼亚移民，1946年与保罗·乔布斯结婚，1955年领养刚出生不久的史蒂夫·乔布斯。

3. **帕蒂·乔布斯**(Patty Jobs)：保罗和克拉拉·乔布斯在领养史蒂夫·乔布斯两年后领养的女儿。

### 生父母及亲妹

4. **阿卜杜法塔赫·约翰·钱德里**(Abdulfattah "John" Jandali)：出生于叙利亚，威斯康星大学硕士，乔布斯和莫娜·辛普森的生父；后来在雷诺市附近的布姆顿赌场担任餐饮经理。

5. **乔安妮·希贝尔·钱德里·辛普森**(Joanne Schieble Jandali Simpson)：出生于威斯康星州，史蒂夫·乔布斯的生母，把乔布斯送养，后来生了女儿莫娜·辛普森，将她抚养成人。

6. **莫娜·辛普森**(Mona Simpson)：乔布斯的亲妹妹，1986年两人相认后，手足之情日益加深。她的几部小说均以家人为原型，例如写母亲乔安妮的《四海为家》(Anywhere but Here)，刻画乔布斯和丽萨父女关系的《凡夫俗子》(A Regular Guy)，还有描述父亲阿卜杜法塔赫·钱德里的《消失的父亲》(The Lost Father)。

## 二、配偶与子女

1. **克里斯安·布伦南**(Chrisann Brennan)：乔布斯在霍姆斯特德中学的女朋友，乔布斯的女儿丽萨(Lisa)的母亲。

2. **丽萨·布伦南-乔布斯**(Lisa Brennan-Jobs)：乔布斯和克里斯安·布伦南之女，出生于1978年，作家，居于纽约。

3. **劳伦娜·鲍威尔**(Laurene Powell)：头脑聪明，亲切随和，毕业于宾夕法尼亚大学，曾在高盛工作，后来进入斯坦福商学院，1991年与史蒂夫·乔布斯结婚。

4. **里德·乔布斯**(Reed Jobs)：史蒂夫·乔布斯和劳伦娜·鲍威尔的长子。

5. **艾琳·乔布斯**(Erin Jobs)：史蒂夫·乔布斯和劳伦娜·鲍威尔的次女。

6. **伊芙·乔布斯**(Eve Jobs)：乔布斯和劳伦娜最小的孩子。

## 三、早期影响者与密友

### 青年时期

1. **乙川弘文**(Kobun Chino)：加利福尼亚州曹洞宗禅师，乔布斯的灵魂导师。

2. **丹尼尔·科特基**(Daniel Kottke)：乔布斯在里德学院的至交，两个人曾相伴踏上印度朝圣之旅，苹果初创时期员工。

3. **伊丽莎白·霍姆斯**(Elizabeth Holmes)：丹尼尔·科特基就读里德学院时期的女朋友，苹果创业初期员工。

4. **罗伯特·弗里德兰**(Robert Friedland)：里德学院学生，拥有一家苹果农场公社，寻求精神启蒙，对乔布斯影响很大；后来经营了一家矿产公司。

5. **史蒂芬·沃兹尼亚克**(Stephen Wozniak)：霍姆斯特德中学的小名人，电子极客，设计的电路板极为出色。乔布斯对他设计的电路板进行包装和营销，二人共同创立了苹果。

### 雅达利时期

1. **阿尔·奥尔康**(Al Alcorn)：雅达利公司总工程师，街机电子游戏《乓》(Pong)的设计者，乔布斯年轻时的老板。

2. **诺兰·布什内尔**(Nolan Bushnell)：雅达利公司创始人，乔布斯的创业偶像。

3. **罗恩·韦恩**(Ron Wayne)：在雅达利与乔布斯相识，在乔布斯和沃兹尼亚克创立苹果之初，成为首个合伙人，后来不明智地放弃了苹果的股权。

## 四、苹果公司核心人物

### 早期创业团队（上市前）

1. **罗德·霍尔特**(Rod Holt)：烟不离手的马克思主义者，1976年被乔布斯聘用为Apple II（第二代苹果电脑）的电子工程师。

2. **德尔·约克姆**(Del Yocam)：苹果早期员工，Apple II团队总经理，后来担任苹果首席运营官。

3. **比尔·阿特金森**(Bill Atkinson)：苹果早期员工，为Macintosh（即Mac）开发图形界面。

### 管理层与董事会

1. **迈克·马库拉**(Mike Markkula)：苹果的首位大股东和董事长，对乔布斯而言是父亲般的人物。

2. **迈克·斯科特**(Mike Scott)：1977年被马库拉聘为苹果总裁，工作职责是管束乔布斯。

3. **约翰·斯卡利**(John Sculley)：百事前总裁，1983年被乔布斯聘请担任苹果首席执行官，最后与乔布斯决裂，于1985年把乔布斯赶出苹果。

4. **吉尔·阿梅里奥**(Gil Amelio)：1996年成为苹果首席执行官，任职期间收购NeXT，让乔布斯重回苹果。

5. **蒂姆·库克**：个性沉稳冷静，1998年被乔布斯聘用为首席运营官，2011年8月接替乔布斯成为苹果的首席执行官。

6. **阿瑟·洛克**(Arthur Rock)：科技投资界传奇，早期苹果董事会成员，被乔布斯视为父亲般的人物。

7. **比尔·坎贝尔**(Bill Campbell)：乔布斯在苹果第一次任职期间的市场营销主管；1997年乔布斯回归后，成为苹果董事会成员和乔布斯的亲信。

### Mac团队核心成员

1. **安迪·赫兹菲尔德**(Andy Hertzfeld)：软件工程师，为人友善，爱开玩笑，早期在Mac团队跟乔布斯一起打拼。

2. **乔安娜·霍夫曼**(Joanna Hoffman)：早期Mac团队成员，敢于对抗乔布斯。

3. **黛比·科尔曼**(Deborah "Debi" Coleman)：早期Mac团队经理，后来接管苹果制造部门。

4. **伯勒尔·史密斯**(Burrell Smith)：Mac团队创始成员，天资聪颖，20世纪90年代患上精神分裂症，饱受折磨。

5. **迈克·默里**(Mike Murray)：Mac电脑早期市场营销总监。

### 产品与技术负责人

1. **乔纳森·艾夫**(Jonathan "Jony" Ive)：苹果首席设计师，乔布斯的合作伙伴和至交。

2. **托尼·法德尔**(Tony Fadell)：朋克工程师，2001年加入苹果开发iPod。

3. **斯科特·福斯托**(Scott Forstall)：苹果移动设备软件负责人。

4. **乔纳森·鲁宾斯坦**(Jonathan "Ruby" Rubinstein)：乔布斯在NeXT的工作伙伴，1997年成为苹果首席硬件工程师，大家常叫他鲁比。

5. **阿维·泰瓦尼安**(Avadis "Avie" Tevanian)：在NeXT跟乔布斯和鲁比共事，1997年成为苹果首席软件工程师。

6. **埃迪·库伊**(Eddy Cue)：苹果互联网服务负责人，是乔布斯与内容供应商洽谈时的得力助手。

### 零售与运营

1. **罗恩·约翰逊**(Ron Johnson)：2000年被乔布斯招致麾下，负责苹果零售店的运营。

### 市场营销与公关

1. **里吉斯·麦肯纳**(Regis McKenna)：公关奇才，从乔布斯创业之初就一直提供指导意见，乔布斯自始至终对他信任有加。

2. **李·克劳**(Lee Clow)：广告鬼才，制作了苹果的广告"1984"，与乔布斯合作长达30年。

3. **安迪·坎宁安**(Andrea "Andy" Cunningham)：里吉斯·麦肯纳公司的公关人员，在早期Mac年代负责苹果的公关事务。

4. **詹姆斯·文森特**(James Vincent)：热爱音乐的英国人，苹果广告代理公司合伙人中较为年轻的一位，与李·克劳和邓肯·米尔纳(Duncan Milner)共事。

5. **丹·列文**(Dan'L Lewin)：跟随乔布斯，相继在苹果和NeXT担任市场营销高管。

## 五、皮克斯/迪士尼相关

1. **阿尔维·雷·史密斯**(Alvy Ray Smith)：皮克斯联合创始人，后与乔布斯发生矛盾。

2. **罗伯特·艾格**(Robert Iger)：2005年接替艾斯纳成为迪士尼首席执行官。

3. **埃德温·卡特穆尔**(Edwin Catmull)：皮克斯联合创始人，后成为迪士尼高管。

4. **迈克尔·艾斯纳**(Michael Eisner)：迪士尼首席执行官，性格强硬，达成与皮克斯的交易，后与乔布斯爆发冲突。

5. **杰弗瑞·卡森伯格**(Jeffrey Katzenberg)：迪士尼电影部门负责人，跟艾斯纳闹翻，于1994年辞职。梦工厂联合创始人。

6. **约翰·拉塞特**(John Lasseter)：皮克斯联合创始人，创意主力。

## 六、业界关系与合作伙伴

1. **保罗·欧德宁**(Paul Otellini)：英特尔首席执行官，协助Mac改用英特尔芯片，但是没拿到iPhone项目。

2. **拉里·埃里森**(Larry Ellison)：甲骨文首席执行官，乔布斯的密友。

3. **比尔·盖茨**：另一位出生于1955年的电脑奇才。

4. **让-路易·加西**(Jean-Louis Gassée)：苹果法国分公司经理，1985年乔布斯被驱逐出苹果后，他接管了Mac部门。

5. **艾伦·凯**(Alan Kay)：设想了早期个人计算机的先驱人物，充满创意，个性有趣，协助安排乔布斯参观施乐的帕洛阿尔托研究中心，协助乔布斯收购皮克斯。

## 七、法律与顾问

1. **乔治·莱利**(George Riley)：出生于田纳西州孟菲斯市，乔布斯的朋友兼律师。
